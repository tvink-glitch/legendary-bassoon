# Weather App

Навчальний курсовий проєкт з розробки консольного програмного продукту.

## Опис
Програма призначена для отримання та відображення поточної погоди в містах України.

## Технології
- Мова програмування: Python
- API: OpenWeather

## Запуск
1. Встановити залежності:
pip install requests

2. Вказати API-ключ у файлі weather.py

3. Запустити програму:
python main.py
